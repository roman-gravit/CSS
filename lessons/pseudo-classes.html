Examples

    

        

            link-red element that has not yet been visited
            visited-blue element that has been visited
            any-link  independent of whether link has been visited (a or area)
            active-grey being activated by the user
            hover-yellow when the user hovers over an element with the cursor (mouse pointer)
            empty-red represents any element that has no children and no text
